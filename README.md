## About

The Sensorium Project treats the dancing body as both instrument and sensor array. Working with soft fluidic pressure sensors, micro-vibration elements, and wearable interfaces, we ask how movement, pressure, and touch can be translated into light, sound, and texture without flattening the complexity of feeling.

**Central Question:** *What happens when a dancing body, soft robotics, and fluidic sensors begin to think and feel together?*
